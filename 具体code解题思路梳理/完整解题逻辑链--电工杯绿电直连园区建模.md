# 第一问 — 基准场景：固定运行下的指标标定

## 1.1 问题的数学本质

**核心问题**：在设备**满负荷、24小时连续运行**的前提下，给定风光出力数据（附件2典型场景），计算园区的运行指标。

这本质上是一个**零自由度、纯解析计算**问题——没有任何决策变量，所有量都由功率平衡方程直接导出。

## 1.2 数学模型逐层展开

### 第一层：功率平衡方程

系统在任意时段 $t$（$t = 0, 1, \ldots, 23$）满足能量守恒：

$$\underbrace{P_{\text{wind}}(t)}_{\text{风电出力}} + \underbrace{P_{\text{solar}}(t)}_{\text{光伏出力}} + \underbrace{P_{\text{buy}}(t)}_{\text{网购电功率}} = \underbrace{P_{\text{sell}}(t)}_{\text{上网功率}} + \underbrace{P_{\text{eq}}}_{\text{设备总功率}} + \underbrace{P_{\text{load}}(t)}_{\text{常规负荷}}$$

其中，设备总功率为**固定常数**：

$$P_{\text{eq}} = 10\ (\text{ALKEL}) + 10\ (\text{PEMEL}) + 0.75\ (\text{NH}_3) = 20.75\ \text{MW}$$

### 第二层：购售电的解析决策

由于没有调节自由度，购售电由净功率的符号唯一决定。定义**净功率**：

$$P_{\text{net}}(t) = P_{\text{wind}}(t) + P_{\text{solar}}(t) - P_{\text{eq}} - P_{\text{load}}(t)$$

则购售电分离：

$$\boxed{P_{\text{buy}}(t) = \max(-P_{\text{net}}(t), 0), \quad P_{\text{sell}}(t) = \max(P_{\text{net}}(t), 0)}$$

这个公式的物理含义非常直觉化：
- $P_{\text{net}}(t) < 0$ → 风光不够用 → 差额从电网购买
- $P_{\text{net}}(t) > 0$ → 风光有盈余 → 多余电力卖回电网
- 两者**互斥**，同一时刻不可能既买又卖

### 第三层：绿电直连三指标

依据政策文件，三个核心指标的定义：

$$\eta_1 = \frac{E_{\text{load}}^{\text{total}} - E_{\text{sell}} - E_{\text{buy}}}{E_{\text{RE}}} \quad (>60\%)$$

$$\eta_2 = \frac{E_{\text{RE}} - E_{\text{sell}}}{E_{\text{load}}^{\text{total}}} \quad (>30\%)$$

$$\eta_3 = \frac{E_{\text{sell}}}{E_{\text{RE}}} \quad (<20\%)$$

三个指标的物理含义：
- **$\eta_1$（自发自用率）**：风光发电有多少真正被园区自己用了
- **$\eta_2$（绿电比例）**：园区总用电中绿色电力占多少
- **$\eta_3$（上网比例）**：风光发电有多少卖回电网了（越高越依赖电网消纳）

### 第四层：吨氨成本核算

这是一个**后评估计算**，不是优化目标。总日成本包含：

$$\text{总成本} = \underbrace{\sum_t P_{\text{buy}}(t) \cdot c_t^{\text{TOU}} \cdot 1000}_{\text{购电费（分时电价）}} + \underbrace{E_{\text{wind}} \cdot 0.15 + E_{\text{solar}} \cdot 0.12}_{\text{风光度电成本}} + \underbrace{P_{\text{ALKEL}} \cdot 0.10 + P_{\text{PEMEL}} \cdot 0.15}_{\text{拆分运维费（v2改进）}} + \underbrace{P_{\text{NH3}} \cdot 0.002}_{\text{合成氨运维}} + \underbrace{\text{DEP}}_{\text{折旧}} - \underbrace{\sum_t P_{\text{sell}}(t) \cdot 0.3779 \cdot 1000}_{\text{售电收入}}$$

吨氨成本：

$$C_{\text{ton}} = \frac{\text{总成本}}{36\ \text{吨}}$$

> **v2改进的意义**：将ALKEL和PEMEL的运维费从混合平均值拆分为独立核算（0.10 vs 0.15 元/kWh），精确反映两种电解槽的效率差异。

## 1.3 求解方法与关键结果

**求解方法**：纯解析计算，Python + NumPy 逐时代入即可，无需任何优化求解器。

关键结果：

| 指标                |    数值     |       结论        |
| ------------------- | :---------: | :---------------: |
| $\eta_1$ 自发自用率 |  **28.2%**  | ❌ 不合格（< 60%） |
| $\eta_2$ 绿电比例   |    69.2%    |      ✅ 合格       |
| $\eta_3$ 上网比例   |  **35.9%**  | ❌ 不合格（> 20%） |
| 吨氨成本            | ~4322 元/吨 |         —         |

### 不达标的根本原因

**时间错配**：光伏在正午时段大发（最高约50 MW），远超园区20.75 MW的用电需求，大量余电被迫上网（拉低 $\eta_1$，推高 $\eta_3$）；夜间/弱风时段风光不足，又需网购电补充（进一步降低 $\eta_1$）。

## 1.4 Q1在整体研究中的位置

Q1建立了**基准线**（baseline）。它揭示的核心矛盾——**设备运行刚性 vs 风光出力波动性**——直接引出了Q2/Q3的调节优化方向。

# 第二问 — 离散调节：0/1开关下的MILP优化

## 2.1 从Q1到Q2的逻辑递进

Q1揭示了"固定运行导致绿电指标不达标"。Q2的第一个改进方向：**让设备可以关停**——风光不足时停机、风光充足时开机。但设备只能"全额开"或"全额关"（离散0/1）。

另外，产能扩容至 **72吨/日**（2×基准），产量分5档（72/63/54/45/36吨/天）。

## 2.2 MILP模型结构

### 决策变量（每时段）

| 变量            |         类型         | 含义                           |
| --------------- | :------------------: | ------------------------------ |
| $x_t$           |      $\{0, 1\}$      | 设备运行/停机                  |
| $\text{buy}_t$  | $\mathbb{R}_{\ge 0}$ | 网购电功率 (MW)                |
| $\text{sell}_t$ | $\mathbb{R}_{\ge 0}$ | 上网电功率 (MW)                |
| $g_t$           |      $\{0, 1\}$      | 电网方向指示（1=售电, 0=购电） |

> 注：Q2采用**聚合设备建模**——设备总功率41.5 MW作为一个整体来开关。

### 约束条件

**约束①：运行时长约束**

$$\sum_{t=0}^{23} x_t = \left\lceil \frac{\text{日产量目标}}{3.0\ \text{ton/h}} \right\rceil$$

含义：满负荷产氨速率3.0 ton/h，日产量目标 ÷ 3.0 = 需要的满负荷运行小时数，向上取整。

**约束②：功率平衡**

$$P_{\text{wind}}(t) + P_{\text{solar}}(t) + \text{buy}_t = \text{sell}_t + x_t \cdot 41.5 + P_{\text{load}}(t)$$

当 $x_t = 0$ 时，设备完全停机；当 $x_t = 1$ 时，设备满功率运行。

**约束③：购售电互斥（Big-M法）**

$$\text{buy}_t \le M(1 - g_t), \quad \text{sell}_t \le M \cdot g_t$$

$M = 104$ MW（风光总装机容量），足以覆盖所有功率场景。

### 目标函数

$$\min \sum_t \Big[\text{buy}_t \cdot c_t^{\text{TOU}} \cdot 1000 + x_t \cdot 41.5 \cdot 1000 \cdot c_{\text{om}} - \text{sell}_t \cdot 0.3779 \cdot 1000\Big] + \underbrace{P_{\text{green}}}_{\text{绿电惩罚}}$$

### v2改进：绿电指标软约束

不把绿电指标作为硬约束（否则可能无可行解），而是引入**惩罚函数**：

$$\begin{aligned} s_{\text{self}} &\ge 0.60E_{\text{RE}} - (E_{\text{load}} - E_{\text{sell}} - E_{\text{buy}}) \\ s_{\text{green}} &\ge 0.30E_{\text{load}} - (E_{\text{RE}} - E_{\text{sell}}) \\ s_{\text{feed}} &\ge E_{\text{sell}} - 0.20E_{\text{RE}} \end{aligned}$$

三个松弛变量 $s_{\text{self}}, s_{\text{green}}, s_{\text{feed}} \ge 0$ 度量偏离目标的程度。

惩罚项：

$$P_{\text{green}} = \underbrace{0.5}_{\lambda_1} s_{\text{self}} + \underbrace{0.3}_{\lambda_2} s_{\text{green}} + \underbrace{1.5}_{\lambda_3} s_{\text{feed}}$$

**$\lambda_3=1.5$ 最高**的原因：上网比例超标意味着绿电外送套利，惩罚最重以抑制该行为。

## 2.3 求解规模与策略

- 求解器：PuLP + CBC
- 规模：48个二元变量（24时段 × 2类二元变量）
- 求解次数：典型场景1次 + 24场景 × 5产量 = **120次MILP**
- 每次耗时：约0.1–0.5秒

## 2.4 关键结果与局限

| 指标             |        数值         |
| ---------------- | :-----------------: |
| 最优日产量       | **72吨/天**（满产） |
| 典型场景吨氨成本 |     ~5447 元/吨     |
| 全年加权平均     |     ~5943 元/吨     |
| 全满足天数占比   |   ~33%（~120天）    |
| 求解成功率       |   120/120 (100%)    |

### 离散调节的核心局限

设备只能全开/全关 → 无法在高电价时段降功率运行 → 购电成本难以进一步压缩。

这自然引出Q3的改进方向：**允许设备功率连续可调**。

# 第三问 — 连续调节：拆分设备建模的MILP

## 3.1 Q2→Q3的核心飞跃

| 维度     | Q2（离散）  |                  Q3（连续）                  |
| -------- | :---------: | :------------------------------------------: |
| 设备建模 | 聚合41.5 MW | **拆分**：ALK(20MW) + PEM(20MW) + NH₃(1.5MW) |
| 运行状态 | 0/1全开全关 |        **半连续**：各设备独立下限约束        |
| 决策粒度 |  48 binary  |          72 binary + 144 continuous          |
| 运维核算 | 按运行小时  |         **按实际用电量**（独立费率）         |
| H₂平衡   |  间接保证   |              **逐时不等式约束**              |
| NH₃爬坡  |     无      |            **硬约束 ±0.30 MW/h**             |

### 这是v2版本最重要的结构改进

v1将ALK、PEM、NH₃合并为一个"总设备功率"变量，而v2将三类设备**独立建模**：

$$\begin{aligned} p_{\text{alk}}[t] &\in [0, 20]\ \text{MW} \\ p_{\text{pem}}[t] &\in [0, 20]\ \text{MW} \\ p_{\text{nh3}}[t] &\in [0, 1.65]\ \text{MW} \end{aligned}$$

## 3.2 约束体系（v2增强版）

### 约束①：半连续约束（per-device, per-hour）

$$\begin{aligned} \text{P\_MIN\_ALK} \cdot z_{\text{alk}}[t] \le p_{\text{alk}}[t] &\le 20 \cdot z_{\text{alk}}[t] \\ \text{P\_MIN\_PEM} \cdot z_{\text{pem}}[t] \le p_{\text{pem}}[t] &\le 20 \cdot z_{\text{pem}}[t] \\ \text{P\_MIN\_NH3} \cdot z_{\text{nh3}}[t] \le p_{\text{nh3}}[t] &\le 1.65 \cdot z_{\text{nh3}}[t] \end{aligned}$$

其中：
- $\text{P\_MIN\_ALK} = 2.0\ \text{MW}$（10%最低负荷）
- $\text{P\_MIN\_PEM} = 0\ \text{MW}$（无最低限制 → **灵活度最高**）
- $\text{P\_MIN\_NH3} = 0.15\ \text{MW}$（10%最低负荷）

> **v3→v4关键修正**：ALK最低负荷从26%恢复为10%，扩大了低风光时段的运行窗口。

### 约束②：产量约束

$$\sum_{t=0}^{23} p_{\text{nh3}}[t] \times 0.002 \times 1000 = \text{日产量目标}$$

其中 $0.002\ \text{ton/(kW·h)}$ 是NH₃的电-氨转换系数。

### 约束③：H₂逐时平衡（v2新增，最关键的新约束）

$$p_{\text{alk}}[t] \times 14 + p_{\text{pem}}[t] \times 16 \ge p_{\text{nh3}}[t] \times 400 \quad \forall t$$

- ALK产氢效率：14 kg/(h·MW)
- PEM产氢效率：16 kg/(h·MW)
- NH₃耗氢率：400 kg/(MW·h)

这个约束确保了**每个时段**的氢供需平衡，防止"超前制氢"的不合理调度。

> **PEM优先效应**：因为PEM效率更高（16 > 14 kg/MW），模型自动优先使用PEM制氢。验证结果：W0S0场景中PEM利用率95% vs ALK利用率53%——数学优化自然选择了更高效的路径。

### 约束④：NH₃爬坡约束（v2新增）

$$|\Delta p_{\text{nh3}}[t]| \le 0.30\ \text{MW/h} \quad \forall t \ge 1$$

即合成氨装置功率变化速率不能超过20%/h（相对于1.5 MW基准）。

### 约束⑤：功率平衡 + 购售电互斥

$$P_{\text{wind}}[t] + P_{\text{solar}}[t] + \text{buy}[t] = \text{sell}[t] + p_{\text{alk}}[t] + p_{\text{pem}}[t] + p_{\text{nh3}}[t] + P_{\text{load}}[t]$$

$$\text{buy}[t] \le 1000(1 - g_t), \quad \text{sell}[t] \le 1000 \cdot g_t$$

## 3.3 目标函数（v2拆分运维）

$$\min \underbrace{\sum_t \text{buy}[t] \cdot c_t^{\text{TOU}} \cdot 1000}_{\text{购电费}} + \underbrace{\sum_t (p_{\text{alk}}[t] \cdot 0.10 + p_{\text{pem}}[t] \cdot 0.15 + p_{\text{nh3}}[t] \cdot 0.002) \cdot 1000}_{\text{拆分运维费}} - \underbrace{\sum_t \text{sell}[t] \cdot 0.3779 \cdot 1000}_{\text{售电收入}} + P_{\text{green}}$$

关键差异：Q2运维费按"运行小时 × 总功率 × 平均费率"，Q3按"实际用电量 × 各设备独立费率"——后者精确得多。

## 3.4 求解规模与校验

- 每实例约210变量 + 280约束
- 0.1–0.3秒/MILP
- **六层校验**：半连续、功率平衡、产量达标、H₂平衡、NH₃爬坡、绿电指标范围

## 3.5 关键结果：Q3 vs Q2

| 指标           | Q2（离散） |    Q3（连续）     |   改善    |
| -------------- | :--------: | :---------------: | :-------: |
| W0S0 72t/d成本 |    5447    |    **5384.50**    | **−1.1%** |
| 更优场景占比   |     —      |  106/120 (88.3%)  |     —     |
| 平均改善       |     —      | **−267.73 元/吨** |     —     |
| 年度加权平均   |  5942.56   |    **5720.40**    |   −3.7%   |

### 降幅分布规律

- **满产72t/d**：Q3 ≈ Q2（24h满功率 → 几乎无灵活空间）
- **中低产量**：降幅显著 → 谷时多运行（0.3424元/kWh）、峰时降功率（0.8024元/kWh）
- **极端场景**：最大降幅10.6%

# 第四问 — 离网运行与储能配置

## 4.1 从联网到离网的范式转换

前三问都有电网作为"无限容量的缓冲池"。Q4切断电网连接（$\text{buy}[t] = \text{sell}[t] = 0$），园区必须仅靠风光+储能自给自足。

## 4.2 第一部分：无储能离网

### 核心原则

"尽限利用风光发电" → 可用功率全部投入制氨。纯计算，无优化。

### 逐时计算逻辑

$$\begin{aligned} P_{\text{avail}}(t) &= \max(P_{\text{wind}}(t) + P_{\text{solar}}(t) - P_{\text{load}}(t), 0) \\ p(t) &= \begin{cases} \min(P_{\text{avail}}(t), P_{\text{max}}) & P_{\text{avail}}(t) \ge P_{\text{min}} \\ 0 & \text{otherwise} \end{cases} \\ \text{curt}(t) &= \max(P_{\text{avail}}(t) - P_{\text{max}}, 0) \end{aligned}$$

其中 $P_{\text{max}} = 41.5$ MW，$P_{\text{min}} = 2.15$ MW（ALK=2.0 + PEM=0 + NH₃=0.15）。

### 关键发现

**离网无储能不可行**：最差场景（W1S3）日产量仅 **7.37吨/天**（目标72吨），即使将风光装机缩小系数α从1.0递减，也无法在全部24场景下达到36吨/天。

## 4.3 第二部分：储能优化配置

### 策略

1. 识别最大弃电场景（W3S0，弃电173.7 MWh/天）
2. 为该场景构建储能MILP
3. 固定最优配置推广至全部24场景

### 储能MILP模型

**SOC动态方程**（考虑自损耗和充放电效率）：

$$\text{SOC}[t+1] = \text{SOC}[t] \times 0.998 + 0.9 \cdot P_c[t] - P_d[t] / 0.9$$

- 自损耗：0.2%/h → 乘0.998
- 充电效率：90%
- 放电效率：90%
- 往返效率：$0.9 \times 0.9 \approx 81\%$

**约束**：
- SOC范围：$0.1C \le \text{SOC}[t] \le 0.9C$
- 充放电功率：$P_c[t], P_d[t] \le P_s$
- 日循环闭合：$\text{SOC}[24] = \text{SOC}[0] = 0.5C$
- 充放电互斥

**目标函数**：

$$\min\ C \times 1000\ /\ 15\ /\ 365$$

即最小化储能容量的日折旧（15年直线折旧）。

### 求解过程

72吨/天不可行（受 $P_{\text{max}}=41.5$ MW约束） → 以2吨/天步长递减 → **56吨/天找到可行解**。

## 4.4 第三部分：经济性对比

| 维度     |          离网+储能 | 并网（无储能） |
| :------- | -----------------: | :------------: |
| 年产量   |           9,536 吨 |   12,960 吨    |
| 年总成本 |      5,152.35 万元 | 5,714.39 万元  |
| 吨氨成本 | **5,403.06 元/吨** | 4,409.25 元/吨 |
| 储能投资 |          1.86 亿元 |       0        |

**离网溢价 = +993.81 元/吨（+22.5%）**

### 离网经济性劣势的根源

1. **储能摊销是核心成本**：186.36 MWh × 1000元/kWh = 1.86亿元 → 日折旧34,039元 → 每吨氨摊1,303元储能成本，占总吨成本24%
2. **离网产量受限**：$P_{\text{max}}=41.5$ MW限制了大发时段的产能上限
3. **并网同样受益于P_min降低**：低P_min使并网系统减少高价购电

> **核心结论**：降低设备下限改善了运行灵活性，但离网经济性的**真正瓶颈是储能投资成本**，而非设备运行约束。

# 第五问 — 综合分析：高渗透率影响与政策建议

## 5.1 方法论：三步法

```
提取论据（Q1–Q4定量数据）
    ↓
因果分析（电力系统原理推演）
    ↓
政策建议（附论证依据）
```

## 5.2 三利三弊的逻辑结构

### 三利（高渗透率的正面效益）

| 利                     | 核心逻辑                                 | 定量支撑                                                   |
| :--------------------- | ---------------------------------------- | ---------------------------------------------------------- |

| **削峰填谷降调峰压力** | 谷时多用电、峰时少购电甚至反送           | Q3：购电集于谷时(0.3424元/kWh)、售电集于峰时(0.8024元/kWh) |

### 三弊（高渗透率的挑战）

| 弊                  | 核心逻辑                   | 定量支撑                                                   |
| :------------------ | -------------------------- | ---------------------------------------------------------- |
| **频率/电压稳定性** | 风光不可调度、日内剧烈波动 | Q1：光伏0–50MW波动，上网比例35.9%；Q3聚类：场景间成本差47% |

# 五问逻辑总览

```
Q1 基准标定 ──→ 发现问题：时间错配，η₁=28.2%不达标，η₃=35.9%超标
    │
    ▼
Q2 离散调节 ──→ 部分改善，但全开/全关无法避开高电价
    │              最优日产量72吨/天，全年全满足仅33%
    ▼
Q3 连续调节 ──→ 拆分设备建模 + 半连续约束 + H₂逐时平衡
    │              PEM优先效应自动产生，88.3%场景优于Q2
    │              年度加权5720.40元/吨（vs Q2的5942.56）
    ▼
Q4 离网运行 ──→ 无储能不可行 → 储能MILP → 最优配置186.36MWh
    │              离网溢价+22.5%，核心瓶颈是储能投资成本
    ▼
Q5 综合分析 ──→ 三利三弊 + 四条政策建议
                  证据链贯穿Q1–Q4全流程
```

整个建模链条的递进关系可以概括为：

$$ \text{固定} \xrightarrow{\text{引入0/1开关}} \text{离散} \xrightarrow{\text{拆分+连续}} \text{灵活联网最优} \xrightarrow{\text{切断电网}} \text{离网+储能} \xrightarrow{\text{综合}} \text{政策建议} $$

每一步都解决前一步暴露的核心矛盾，构成了一个完整的"发现问题 → 建模优化 → 边界探索 → 政策转化"的研究闭环。
