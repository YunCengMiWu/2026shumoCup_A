## 第一部分：离网无储能逐场景解析

### 1.1 数学模型

离网运行的核心约束：**园区用能全部来自风光，无电网交互**。

功率平衡方程（逐时）：

$$\boxed{P_{\text{wind}}[t] + P_{\text{solar}}[t] = P_{\text{load}}[t] + p[t] + \text{curt}[t] \quad \forall t \in [0, 23]}$$

其中：

- $p[t]$：设备实际用电功率（≥ 0）
- $\text{curt}[t]$：弃电量（风光超出设备消纳能力的部分）

### 1.2 逐时决策逻辑（无优化自由度）

与 Q2/Q3 不同，离网无储能时**没有任何优化自由度**——风光是唯一的能量来源，设备只能在可用功率范围内运行。

**步骤 1**：计算净可用功率

$$P_{\text{net}}[t] = P_{\text{wind}}[t] + P_{\text{solar}}[t] - P_{\text{load}}[t]$$

**步骤 2**：确定设备实际功率

$$p[t] = \begin{cases} 0 & P_{\text{net}}[t] < P_{\min} \\ P_{\text{net}}[t] & P_{\min} \le P_{\text{net}}[t] \le P_{\text{total}} \\ P_{\text{total}} & P_{\text{net}}[t] > P_{\text{total}} \end{cases}$$

其中：

- $P_{\text{total}} = 41.5$ MW（72t/d 满功率）
- $P_{\min}$ = 逐台最低功率之和（见下文）

**步骤 3**：计算弃电量

$$\text{curt}[t] = \max\big(P_{\text{net}}[t] - P_{\text{total}},\ 0\big)$$

代码实现：

```python
for t in range(24):
    net = wind[t] + solar[t] - load[t]
    p_avail = max(net, 0.0)
    if p_avail >= P_MIN:
        p[t] = min(p_avail, P_TOTAL)
    curt[t] = max(net - P_TOTAL, 0.0)
```

### 1.3 $P_{\min}$ 的推导（逐台最小值之和）

Q4 使用聚合变量 $p[t]$（ALK+PEM+NH₃ 总功率），因此 $P_{\min}$ 采用各设备最低功率之和：

$$\boxed{P_{\min} = \underbrace{0.10 \times 20}_{\text{ALK min}} + \underbrace{0.00 \times 20}_{\text{PEM min}} + \underbrace{0.10 \times 1.5}_{\text{NH₃ min}} = 2.0 + 0 + 0.15 = 2.15\ \text{MW}}$$

物理含义：$p[t] < 2.15$ MW 时无法同时满足 ALK≥2.0 MW 和 NH₃≥0.15 MW，设备必须停机 → $p[t] = 0$。

### 1.4 日产量计算

$$\boxed{Q_{\text{daily}} = \sum_{t=0}^{23} p[t] \times \frac{\text{NH}_3\text{RATE}_{72}}{P_{\text{total}}} = \sum_{t=0}^{23} p[t] \times \frac{3.0}{41.5}\ \text{ton}}$$

推导逻辑：满功率 ($41.5$ MW) 运行 1 小时产氨 $3.0$ 吨，产率 = $3.0/41.5$ ton/(MW·h)。因此 $p[t]$ 功率运行 1 小时产氨 $p[t] \times 3.0/41.5$ 吨。

### 1.5 吨氨成本（离网版本）

离网时 $E_{\text{buy}} = 0$，$E_{\text{sell}} = 0$：

$$\boxed{C_{\text{ton}} = \frac{\underbrace{E_{\text{wind}} \cdot 0.15 + E_{\text{solar}} \cdot 0.12}_{\text{风光度电成本}} + \underbrace{E_{\text{el}} \cdot \bar{r}_{\text{el}} \cdot 0.125 + E_{\text{el}} \cdot r_{\text{NH}_3} \cdot 0.002}_{\text{拆分运维}} + \underbrace{D_{\text{dep}}}_{\text{设备折旧}}}{Q_{\text{daily}}}}$$

其中运维费按额定功率比例拆分：

- 电解比例：$r_{\text{el}} = 40/41.5 \approx 0.964$
- 合成氨比例：$r_{\text{NH}_3} = 1.5/41.5 \approx 0.036$

## 1.6 最小风光容量估算

**目标**：在所有 24 个场景下，日产量均不低于 36 吨/天，求最小风光装机倍率 $\alpha$。

**方法**：从 $\alpha = 1.0$（当前装机）递减扫描至 $\alpha = 0.1$（步长 0.05）。

对每个 $\alpha$：

$$P_{\text{total}}^{\alpha} = \alpha \cdot 41.5 \quad \text{MW}$$

$$P_{\min}^{\alpha} = \alpha \cdot P_{\min}$$

$$\text{NH}_3^{\alpha} = \alpha \cdot 3.0 \quad \text{ton/h}$$

遍历 24 场景，计算最差日产量：

$$Q_{\min}(\alpha) = \min_{\text{all 24 scenarios}} Q_{\text{daily}}^{\alpha}$$

当 $Q_{\min}(\alpha) \ge 36$ 吨时停止扫描，输出 $\alpha_{\min}$。

代码：

```python
for alpha_try in np.arange(1.0, 0.05, -0.05):
    alpha = round(alpha_try, 2)
    # ... rescale all parameters ...
    for w, s in all_scenarios:
        # compute daily production
        ...
    if min_daily >= min_prod_target:
        found_alpha = alpha
        break
```

## 第二部分：储能优化配置

### 2.1 问题设置

给定最大弃电场景，固定储能容量/功率为未知变量，构建 LP 模型：

$$\min \quad \underbrace{C_{\text{capacity}} \cdot P_{\text{inv}} \cdot 1000 / L / 365}_{\text{储能日折旧}} + \underbrace{0.01 \cdot P_{\text{storage}}}_{\text{微小功率惩罚}}$$

约束于：

1. 产量 ≥ 目标值（初始 72t/d，不可行则递减）
2. SOC 动力学方程
3. 半连续约束
4. 功率平衡（含弃电变量）

### 2.2 决策变量

| 变量 | 类型 | 含义 | 单位 |
|------|:---:|------|:--:|
| $p[t]$ | Continuous | 设备功率 | MW |
| $P_c[t]$ | Continuous | 充电功率 | MW |
| $P_d[t]$ | Continuous | 放电功率 | MW |
| $\text{curt}[t]$ | Continuous | 弃电功率 | MW |
| $z[t]$ | Binary | 设备运行标志 | — |
| $\text{SOC}[t]$ | Continuous | 储能电量 (25个点) | MWh |
| $C_{\text{cap}}$ | Continuous | 储能容量 | MWh |
| $P_{\text{sto}}$ | Continuous | 储能功率 | MW |
| $z[t]$ | Binary | 半连续标志 | — |

### 2.3 SOC 动力学方程（最关键）

$$\boxed{\text{SOC}[t+1] = \text{SOC}[t] \cdot \underbrace{0.998}_{\text{自放电留存}} + \underbrace{0.90}_{\eta_c} \cdot P_c[t] - \underbrace{\frac{1}{0.90}}_{\eta_d^{-1}} \cdot P_d[t] \quad \forall t \in [0, 23]}$$

物理含义（逐项解释）：

1. **自放电**：每时段留存 99.8%，即损耗 0.2%/h
2. **充电**：$P_c[t]$ MW × 1h × 0.90 效率 = $0.90 P_c[t]$ MWh 有效存储
3. **放电**：放出 $P_d[t]$ MWh 需消耗 $P_d[t]/0.90$ MWh 储能 → 系数 $1.111...$

代码：

```python
INV_ETA_D = 1.0 / ETA_D  # 1/0.90 = 1.111...
for t in range(24):
    prob += (soc[t + 1] == soc[t] * SELF_DISCH + ETA_C * Pc[t] - INV_ETA_D * Pd[t],
             f"SOC_dyn_{t}")
```

### 2.4 SOC 边界

$$\boxed{0.10 \cdot C_{\text{cap}} \le \text{SOC}[t] \le 0.90 \cdot C_{\text{cap}} \quad \forall t}$$

$$\boxed{\text{SOC}[0] = \text{SOC}[24] = 0.50 \cdot C_{\text{cap}}}$$

初始和终值均为 50%：保证日循环可持续，第二日可重复相同调度。

### 2.5 充放电功率约束

$$\boxed{0 \le P_c[t] \le P_{\text{sto}}, \quad 0 \le P_d[t] \le P_{\text{sto}} \quad \forall t}$$

储能功率与容量的倍率约束：

$$\boxed{P_{\text{sto}} \le 4.0 \cdot C_{\text{cap}}}$$

即最大 4C 放电率（15 分钟放空），对应锂电储能的技术参数。

### 2.6 功率平衡（含储能 + 弃电）

$$\boxed{P_{\text{wind}}[t] + P_{\text{solar}}[t] + P_d[t] = P_c[t] + p[t] + P_{\text{load}}[t] + \text{curt}[t] \quad \forall t}$$

> **注意**：弃电变量 $\text{curt}[t] \ge 0$ 是 LP 实现的关键——无弃电变量时，风光超过 $P_{\text{total}}$ 时段无法满足功率平衡，导致不可行。

### 2.7 产量约束

$$\boxed{\sum_{t=0}^{23} p[t] \times \frac{3.0}{41.5} \ge Q_{\text{target}}}$$

### 2.8 求解策略

第一轮尝试满产 72t/d → 若不可行则按 2t/d 递减：

```python
opt_target = 72.0
result, prob = _build_storage_lp(..., opt_target, ...)

while result['status'] != 'Optimal' and opt_target > 10.0:
    opt_target -= 2.0
    result, prob = _build_storage_lp(..., opt_target, ...)
```

### 2.9 储能日折旧计算（修复后含 ×1000）

$$\boxed{D_{\text{sto}} = \frac{C_{\text{cap}} \cdot 1000 \cdot P_{\text{inv}} \cdot 1000}{L \cdot 365} = \frac{C_{\text{cap}}\ [\text{MWh}] \times 1000\ [\text{kWh/MWh}] \times 1000\ [\text{元/kWh}]}{15\ [\text{年}] \times 365\ [\text{天/年}]}}$$

验证：$C_{\text{cap}} = 186$ MWh → $D_{\text{sto}} = 186 \times 1000 \times 1000 / (15 \times 365) = 33,972$ 元/天 ✓

代码中三处添加了 ×1000：

```python
daily_dep = capacity * STORAGE_INVESTMENT * 1000 / STORAGE_LIFETIME / 365
```

### 2.10 推广到全部 24 场景

固定已优化的储能容量/功率（$C_{\text{cap}}^*, P_{\text{sto}}^*$），对每个场景：

1. 以该场景无储能时的产量作为目标
2. 若不可行则递减目标值
3. 计算吨氨成本（含储能折旧）

## 第三部分：经济性对比

### 3.1 离网+储能年度统计

$$\boxed{C_{\text{offgrid}}^{\text{annual}} = \sum_{i=1}^{24} C_{\text{ton}}^{(i)} \cdot Q_{\text{daily}}^{(i)} \cdot 15}$$

$$\boxed{Q_{\text{offgrid}}^{\text{annual}} = \sum_{i=1}^{24} Q_{\text{daily}}^{(i)} \cdot 15}$$

$$\boxed{\bar{C}_{\text{offgrid}} = \frac{C_{\text{offgrid}}^{\text{annual}}}{Q_{\text{offgrid}}^{\text{annual}}}}$$

### 3.2 并网模式年度统计（从 Q3 取数据）

对 Q3 的 120 个结果（24 场景 × 5 产量），每场景取**吨氨成本最低**的产量级别：

$$\boxed{\text{best\_level}(i) = \arg\min_{l \in \{72,63,54,45,36\}} C_{\text{ton}}(i, l)}$$

$$\boxed{C_{\text{grid}}^{\text{annual}} = \sum_{i=1}^{24} C_{\text{ton}}(i, \text{best\_level}(i)) \cdot Q_{\text{daily}}(i, \text{best\_level}(i)) \cdot 15}$$

代码：

```python
best_per_scenario = df_q3.loc[df_q3.groupby('场景')['吨氨成本(元/吨)'].idxmin()]
for _, row in best_per_scenario.iterrows():
    prod = float(row['日产量(吨/天)'])
    cost = float(row['吨氨成本(元/吨)'])
    grid_annual_prod += prod * DAYS_PER_SCENARIO
    grid_annual_cost += cost * prod * DAYS_PER_SCENARIO
```

## 关键结果解读

| 指标 | 离网无储能 | 离网+储能 | 并网(无储能) |
|------|:---:|:---:|:---:|
| 年产量 | ~8,772 吨 | ~9,507 吨 | ~12,960 吨 |
| 吨氨成本 | ~6,797 元/吨 | ~5,414 元/吨 | ~4,306 元/吨 |
| 产能利用率 | ~30% | ~33% | ~45% |
| 最小 $\alpha$ | — | — | 0.35 (P_total≈14.5MW) |

### 核心发现

1. **离网模式产量受限**：无储能时最差场景仅~20吨/天，加储能后提升至~25吨/天，但仍远低于 72 吨/天产能
2. **储能价值**：储能将弃电转化为可调度电力，提升产量约 8.4%，降低吨氨成本约 20.3%
3. **并网经济性更优**：并网吨氨成本比离网+储能低约 25.7%，核心原因是通过网购电实现了接近满产的运行，规模效应摊薄了固定成本
4. **$\alpha_{\min} = 0.35$**：即使风光装机缩至当前的 35%，最差场景下仍可达 36 吨/天，说明当前装机（40MW风+64MW光）在大多数场景下严重过剩
