## 运行说明
所有代码使用 Python 3.x，依赖：
- numpy, pandas（数据处理）
- pulp + CBC solver（MILP/LP求解）
- matplotlib（可视化）
- openpyxl（读取附件Excel）

安装依赖后，在各子目录下运行对应.py文件即可复现论文结果。
