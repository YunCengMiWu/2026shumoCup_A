# 储能设备建模与优化配置总结

## 一、A题给出的储能设备相关计算要求与参数

根据附件6《储能设备和合成氨装置技术参数》，电储能设备的相关参数如下：

| 参数     | 数值        | 说明                           |
| -------- | ----------- | ------------------------------ |
| 投资成本 | 1000 ¥/kWh  | 按储能容量计算的一次性投资     |
| 运维系数 | 0.01 元/kWh | 每吞吐1 kWh电量的运行维护费用  |
| 使用寿命 | 15 年       | 用于投资成本年分摊             |
| 充电效率 | 90%         | 电能转换为化学能的效率         |
| 放电效率 | 90%         | 化学能转换为电能的效率         |
| 自损耗率 | 0.2%        | 每小时静止状态下的能量损失比例 |

题目在问题四中要求：
1. 针对最大弃电场景，给出储能容量最优配置方案；
2. 制定24个场景下有储能参与生产运行功率最优调度方案；
3. 计算各场景的吨氨成本、全年制氨总量和氢氨年平均产能利用率；
4. 分析储能带来的园区能源自给和风光发电利用改善状况。

**未明确给出的参数**（需在建模时做合理假设）：
- 储能额定功率与容量的关系（即充放电倍率）；
- 荷电状态（SOC）的安全运行上下限。

## 二、储能设备充电/放电过程的数学模型

### 2.1 基本物理过程

电储能设备在运行时存在三种状态：**充电**、**放电**、**静置**。

- **充电**：将电能转化为化学能存储，实际存储的能量小于输入电能，损耗由充电效率 \(\eta_{\text{ch}}\) 描述。
- **放电**：将化学能转化为电能释放，实际输出电能小于存储的化学能，损耗由放电效率 \(\eta_{\text{dis}}\) 描述。
- **静置**：即使不进行充放电，储能设备内部仍存在自放电现象，导致能量随时间缓慢损失，由自损耗率 \(\sigma\) 描述。

### 2.2 自损耗率的计算方法

自损耗率 \(\sigma = 0.2\%\) 的含义是：**在单位时间（1小时）内，储能设备因内部化学反应或漏电而损失的能量占当前储能量的比例**。

设 \(SOC(t)\) 为 \(t\) 时刻的储能量（单位：kWh），则静置1小时后的储能量为：

\[
SOC(t+1) = SOC(t) \times (1 - \sigma)
\]

其中 \(\sigma = 0.002\)。

### 2.3 荷电状态（SOC）递推方程

在同时考虑充放电和自损耗的情况下，离散时间（时间间隔 \(\Delta t = 1\) 小时）的SOC递推关系为：

\[
SOC(t) = SOC(t-1) \times (1 - \sigma) + \left( \eta_{\text{ch}} \cdot P_{\text{ch}}(t) - \frac{P_{\text{dis}}(t)}{\eta_{\text{dis}}} \right) \cdot \Delta t
\]

式中：
- \(SOC(t)\) —— \(t\) 时刻末的储能量（kWh）；
- \(P_{\text{ch}}(t)\) —— \(t\) 时段内的充电功率（kW）；
- \(P_{\text{dis}}(t)\) —— \(t\) 时段内的放电功率（kW）；
- \(\eta_{\text{ch}} = 0.90\)，\(\eta_{\text{dis}} = 0.90\)；
- \(\Delta t = 1\) h。

**说明**：充电时，输入电网功率 \(P_{\text{ch}}\) 只有 \(\eta_{\text{ch}}\) 倍转化为化学能存入；放电时，从储能释放的化学能只有 \(\eta_{\text{dis}}\) 倍转化为电能输出。

### 2.4 SOC约束条件

为保证储能设备安全运行并延长使用寿命，需对SOC施加上下限约束：

\[
SOC_{\min} \le SOC(t) \le SOC_{\max}
\]

由于题目未明确给出安全边界，基于工程实践，通常取：
\[
SOC_{\min} = 0.1 E_{\text{rated}}, \quad SOC_{\max} = 0.9 E_{\text{rated}}
\]
其中 \(E_{\text{rated}}\) 为储能额定容量（kWh）。若要求更保守，也可取 \([0, E_{\text{rated}}]\)，但过充过放会加速老化。

此外，还需约束充放电功率不超过设备额定功率 \(P_{\text{rated}}\)：
\[
0 \le P_{\text{ch}}(t) \le P_{\text{rated}}, \quad 0 \le P_{\text{dis}}(t) \le P_{\text{rated}}
\]

### 2.5 额定功率与容量的关系

附件6未提供 \(P_{\text{rated}}\) 与 \(E_{\text{rated}}\) 的对应关系。本题采用常见假设：**储能系统的额定功率等于其额定容量除以2小时**，即 0.5C 放电倍率：

\[
P_{\text{rated}} = \frac{E_{\text{rated}}}{2\ \text{h}}
\]

该假设在论文中需明确声明，并可进行敏感性分析以验证其合理性。

## 三、第四问中的求解思路

### 3.1 问题分解

第四问包含三个子问题，其中第(2)问要求针对最大弃电场景配置储能，并制定24个场景下的最优调度方案。核心求解思路如下：

1. **识别最大弃电场景**  
   在24种风光出力场景下，首先进行离网无储能的运行仿真（电解槽功率连续可调，下限10%），计算各场景的弃电量，找出弃电总量最大的场景作为“最大弃电场景”。

2. **储能容量优化**  
   针对最大弃电场景，建立以**年化总成本最小**为目标的优化模型，决策变量为储能容量 \(E_{\text{rated}}\) 及逐小时的充放电功率 \(P_{\text{ch}}(t), P_{\text{dis}}(t)\)。目标函数包括：
   - 储能投资年分摊成本；
   - 储能年运维成本；
   - 弃电损失成本（按弃电量 × 上网电价或惩罚电价）；
   - 电解槽及合成氨运行成本。

   通过求解得到最优的 \(E_{\text{rated}}\) 及对应的 \(P_{\text{rated}}\)。

3. **多场景调度验证**  
   将上一步得到的最优储能配置固定，分别对24个风光场景（各持续15天）进行离网带储能的逐时优化调度，计算每个场景的吨氨成本、制氨总量、产能利用率等指标。

4. **改善效果分析**  
   对比配置储能前后（即问题四(1)与(2)的结果），分析能源自给率、风光发电利用率、吨氨成本等指标的变化。

### 3.2 储能参与下的离网功率平衡

离网运行时，无电网交互，功率平衡方程为：

\[
P_{\text{wind}}(t) + P_{\text{pv}}(t) = P_{\text{load}}(t) + P_{\text{ch}}(t) - P_{\text{dis}}(t) + P_{\text{curt}}(t)
\]

其中：
- \(P_{\text{load}}(t)\) —— 电解槽、合成氨及常规负荷的总和；
- \(P_{\text{curt}}(t)\) —— 弃电功率（当发电大于用电且储能已满或无法吸收时产生）。

目标是在各场景下**最小化弃电量**或**最小化综合成本**。

## 四、求解方法

### 4.1 储能容量优化的数学模型

采用**混合整数线性规划**或**枚举 + 线性规划**相结合的方法。

#### 决策变量
- \(E_{\text{rated}}\) —— 储能额定容量（连续变量，kWh）；
- \(P_{\text{ch}}(t), P_{\text{dis}}(t)\) —— 各小时充放电功率（连续变量，kW）；
- \(SOC(t)\) —— 各小时末储能量（连续变量，kWh）；
- \(u_{\text{ch}}(t), u_{\text{dis}}(t)\) —— 二进制变量，防止同时充放电。

#### 目标函数（年化）
\[
\min \left( \frac{C_{\text{inv}} \cdot E_{\text{rated}}}{T_{\text{life}}} + \sum_{t=1}^{24} \left[ C_{\text{O\&M}} \cdot (P_{\text{ch}}(t) + P_{\text{dis}}(t)) + \lambda \cdot P_{\text{curt}}(t) \right] \right)
\]
其中：
- \(C_{\text{inv}} = 1000\) 元/kWh；
- \(T_{\text{life}} = 15\) 年（不计残值，等额折旧）；
- \(C_{\text{O\&M}} = 0.01\) 元/kWh（按充放电吞吐量计）；
- \(\lambda\) —— 弃电惩罚系数，可取上网电价 0.3779 元/kWh。

#### 约束条件
- SOC递推（含自损耗）；
- SOC上下限：\(0.1 E_{\text{rated}} \le SOC(t) \le 0.9 E_{\text{rated}}\)；
- 功率限幅：\(0 \le P_{\text{ch}}(t) \le P_{\text{rated}}\)，\(0 \le P_{\text{dis}}(t) \le P_{\text{rated}}\)，其中 \(P_{\text{rated}} = E_{\text{rated}} / 2\)；
- 充放电互斥：\(u_{\text{ch}}(t) + u_{\text{dis}}(t) \le 1\)，且 \(P_{\text{ch}}(t) \le M u_{\text{ch}}(t)\)，\(P_{\text{dis}}(t) \le M u_{\text{dis}}(t)\)；
- 功率平衡（含弃电变量）；
- 电解槽与合成氨的功率范围及产量约束（同问题三的连续调节模型）。

### 4.2 求解算法

由于 \(E_{\text{rated}}\) 与 \(P_{\text{rated}}\) 呈线性关系，且目标函数中存在 \(E_{\text{rated}}\) 的一次项，该问题为**线性规划**（若忽略二进制互斥变量，也可用线性规划近似）。实际求解步骤：

1. **外层枚举**：对 \(E_{\text{rated}}\) 在一定范围内（如 0 ~ 50 MWh）离散取值，步长取 1 MWh 或更细；
2. **内层线性规划**：对每个固定的 \(E_{\text{rated}}\)，求解该场景下的最优小时级充放电调度（目标为最小化弃电损失 + 运维成本），得到该容量下的最小日成本；
3. **年化比较**：将日成本乘以365（或按场景天数加权），加上投资年分摊，得到年总成本；
4. **选择最优**：年总成本最小的 \(E_{\text{rated}}\) 即为最优储能容量。

**计算工具**：可使用 Python 的 `PuLP`、`Pyomo` 或 MATLAB 的 `linprog`、`intlinprog` 求解。

### 4.3 多场景调度方案制定

获得最优储能配置后，将其作为固定参数，对每个风光场景（24种，各15天）独立运行线性规划，得到该场景下的逐时充放电计划、弃电功率及吨氨成本。最后汇总全年（24×15=360天）的总制氨量、总成本、平均产能利用率等指标。

### 4.4 结果输出

- 最优储能容量 \(E_{\text{rated}}\)（单位：kWh）及对应额定功率 \(P_{\text{rated}}\)；
- 各场景下的调度曲线（充放电功率、SOC轨迹）；
- 全年总吨氨成本、制氨总量、氢氨年平均产能利用率；
- 与无储能离网方案相比的弃电减少率、能源自给率提升幅度。
